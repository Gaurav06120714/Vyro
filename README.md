# Vyro Ecosystem

A suite of local-first, AI-powered productivity apps built for developers.

| App | Description |
|-----|-------------|
| [VyroBrowser](./VyroBrowser) | Privacy-first Electron browser with local AI |
| [VyroCoding](./VyroCoding) | AI-powered code editor |
| [VyroMusic](./VyroMusic) | Local music player |
| [VyroNotes](./VyroNotes) | Smart note-taking app |
| [VyroPortify](./VyroPortify) | Portfolio builder |

## Setup
Run `npm install` inside each app folder to install dependencies.
